## Käytä laajennuksena

Tämä pakettivarasto voidaan lisätä **laajennuksena** MakeCodessa.

* avaa [https://makecode.microbit.org/](https://makecode.microbit.org/)
* valitse **Uusi projekti**
* valitse hammaspyörävalikon kohta **Laajennukset**
* tee haku **https://github.com/eliasmoinen/lemmikkiiiiiiiiiii** ja tuo

## Muokkaa tätä projektia ![Koontiversion tilamerkki](https://github.com/eliasmoinen/lemmikkiiiiiiiiiii/workflows/MakeCode/badge.svg)

Tämän pakettivaraston muokkaamiseksi MakeCodessa.

* avaa [https://makecode.microbit.org/](https://makecode.microbit.org/)
* valitse **Tuo** ja sitten **Tuo URL**
* liitä **https://github.com/eliasmoinen/lemmikkiiiiiiiiiii** ja valitse tuonti
